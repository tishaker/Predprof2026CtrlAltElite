**Анализ поступления**

Веб-приложение для автоматизации анализа конкурсных списков и расчета проходных баллов в вузе.
Цель: Оптимизировать работу приемных комиссий, предоставить абитуриентам актуальную информацию о вероятности поступления.

Функциональность
1. Для администратора/приемной комиссии:
1.1. Авторизация через логин/пароль.
1.2. Загрузка конкурсных списков абитуриентов из CSV-файлов.
1.3. Автоматическое обновление данных при поступлении новых списков.
1.4. Расчет проходных баллов на каждую образовательную программу.
1.5. Визуализация конкурсных списков с фильтрацией и сортировкой.
1.6. Анализ динамики проходных баллов за 4 дня приемной кампании.
1.7. Генерация отчетов в PDF формате с графиками и статистикой.
2. Для абитуриентов (потенциально):
2.1. Просмотр текущих проходных баллов (если добавить публичный доступ).
2.2. Анализ своих шансов на поступление по приоритетам.

Технологии
1. Бэкенд:
1.2. Python Flask (маршрутизация, бизнес-логика).
1.3. QLAlchemy (ORM для работы с БД).
1.4. Flask-Login (управление сессиями и авторизация).
1.5. Pandas (обработка CSV файлов и анализ данных).
1.6. ReportLab (генерация PDF отчетов).
1.7. Matplotlib (создание графиков и гистограмм).
2. Фронтенд:
2.1. JavaScript (динамический интерфейс, фильтрация, сортировка).
2.2. HTML/CSS + Jinja2 (шаблоны страниц).
2.3. Bootstrap (адаптивный дизайн).
3. База данных:
3.1. SQLite (хранение данных абитуриентов, легковесная и быстрая).

Особенности реализации
1. Автоматическая генерация тестовых данных с учетом требований ТЗ.
2. Реализация каскадного алгоритма зачисления с учетом приоритетов абитуриентов.
3. Поддержка русских шрифтов в PDF отчетах через регистрацию системных шрифтов.
4. Валидация CSV файлов перед загрузкой для обеспечения целостности данных.
   
Установка и запуск
1. Скачать проект
2. Установить зависимости: pip install -r requirements.txt
3. Сгенерировать тестовые данные: python csvgen.py
4. Запустить сервер: python app.py
5. Загрузить данные в базу: python uploadall.py (в отдельном окне после запуска сервера)
6. Открыть в браузере: http://localhost:5000
